Android Voice Memo App

Craft a sleek voice memo app that transcribes and organizes your thoughts.

Tech Stack:
  • Expo
  • React Native
  • OpenAI Whisper
  • Firebase for Storage
Resources:
  • Expo Documentation
  • OpenAI Whisper
  • React Native Docs
Estimated Time: 15 hours
Implementation Roadmap:
  1. Set up Expo development environment
  2. Implement basic UI with React Native
  3. Integrate OpenAI Whisper for transcription
  4. Add Firebase backend for storage

Pro Tip: Use Expo's development build to test voice permissions on physical devices early. Mock Whisper API during initial development to avoid rate limits.

Firecracker VM Testing Playground

Build a web interface for spinning up and managing lightweight VMs using Firecracker for rapid testing and development.

Tech Stack:
  • Firecracker VM
  • React
  • Go (for backend)
  • Docker
  • WebSocket
Resources:
  • Firecracker GitHub
  • Firecracker Documentation
  • AWS Firecracker Overview
Estimated Time: 15 hours
Implementation Roadmap:
  1. Set up Firecracker kernel and root filesystem
  2. Create backend API for VM management
  3. Develop React frontend with WebSocket integration
  4. Implement VM launch and termination workflows

Pro Tip: Start with a single pre-configured VM template. Use Docker to simulate Firecracker environment during initial development. Be prepared for low-level networking challenges.

Math Notes App with Excalidraw

Create an interactive math notes application leveraging Excalidraw's drawing capabilities for mathematical notation and diagrams.

Tech Stack:
  • Excalidraw
  • React
  • MathJax/KaTeX
  • Firebase/Supabase
  • WebRTC for Collaboration
Resources:
  • Excalidraw Official Site
  • Excalidraw GitHub
  • MathJax Documentation
Estimated Time: 15 hours
Implementation Roadmap:
  1. Integrate Excalidraw as base drawing component
  2. Add MathJax for LaTeX rendering
  3. Implement persistent storage with Firebase
  4. Build real-time collaboration using WebRTC

Pro Tip: Fork Excalidraw locally to customize math-specific tools. Use browser's localStorage for quick offline support. Plan WebRTC signaling carefully.
